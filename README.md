# Large Smart Battery

Oxygen Not Included mod that adds a larger smart battery with higher capacity, steel construction, runtime tint coloring, and Chinese/English in-game text.

## Features

- Adds `Large Smart Battery` / `大智能电池`
- Capacity: 100 kJ
- Power loss: 6.7 J/s
- Heat generation: 5 kDTU/s
- Materials: steel, plastic, glass
- Uses the original smart battery animation with a runtime tint
- Supports Chinese and English building text

## Build Notes

The project file references local Oxygen Not Included managed assemblies. Update the hint paths in `源代码/BCSmartBattery.csproj` if your game is installed somewhere else.

Only source and metadata are tracked in this repository. Compiled DLLs, temporary files, and experimental animation resources are intentionally ignored.
